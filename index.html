Claude will search the web for real, citable sources: foundational/highly-cited works and current literature from the last few years. Every card lands with a ready-to-use APA 7 reference.

This app calls the Claude API's live web search from inside Claude.ai (or another host that proxies the API for it). If you open this file outside that environment, the search call won't have credentials to run — the manual-entry and drafting tools below still work fully offline.

AI-assisted drafting can still get details wrong. Before submission, verify every reference against the original source (author names, year, volume/issue, page numbers, DOI) and confirm each in-text citation matches an entry in your reference list. Journal rank and citation-tier estimates from "Assess strength" are AI-researched approximations — cross-check them against Scopus, Web of Science, or Google Scholar before using them to justify a source's inclusion.
